# Using CC65
We use CC65 as our assembler and linker, which can be installed [here](https://cc65.github.io/getting-started.html).

## Building
There are several executables we care about:
- `ca65.exe`: The assembler
- `ld65.exe`: The linker
- `cl65.exe`: Combined tool
